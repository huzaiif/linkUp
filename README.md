# LinkUp - Social Media Application

LinkUp is a responsive social media platform built with Node.js and SQLite. It features a modern UI, user authentication, post creation with images, dynamic feeds, and profile management.

## Features

- **Authentication**: User registration and login.
- **Feed**: Dynamic scrolling feed with "Load More" pagination.
- **Posts**: Create posts with text and images. Delete your own posts.
- **Profile**: User profiles showing only their specific posts.
- **Likes & Comments**: Interactive post engagement.
- **Dark Mode**: Built-in dark/light theme toggle.
- **Responsive Design**: Mobile-friendly layout.

## Tech Stack

- **Frontend**: HTML, CSS (Vanilla), JavaScript
- **Backend**: Node.js, Express
- **Database**: SQLite
- **Deployment**: Ready for Render or Glitch

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/huzaiif/linkUp.git
    cd linkUp
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the server:
    ```bash
    npm start
    ```
4.  Open `http://localhost:3000` in your browser.
